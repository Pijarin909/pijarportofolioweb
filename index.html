Project 1

Ini adalah project team yang saya buat bersama teman-teman saya, Remote Control Car berbasis ESP8266 adalah sebuah mobil miniatur yang dapat dikendalikan melalui jaringan WiFi dengan menggunakan antarmuka web. Mobil dilengkapi dengan motor DC sebagai penggerak roda, motor driver (H-Bridge) sebagai pengatur arah putaran motor, serta sumber daya baterai sebagai catu daya utama. Kendali dilakukan melalui web server sederhana yang dijalankan langsung di ESP8266. Saat perangkat (laptop atau smartphone) terhubung ke jaringan WiFi ESP8266, pengguna dapat mengakses antarmuka kendali dengan mengetikkan alamat IP ESP8266 pada browser.
